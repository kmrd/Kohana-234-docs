Class ORM_Core

Description
Description | Vars (details) | Methods (details)

[Object Relational Mapping][ref-orm] (ORM) is a method of abstracting database access to standard PHP calls. All table rows are represented as model objects, with object properties representing row data. ORM in Kohana generally follows the [Active Record][ref-act] pattern.

[ref-orm]: http://wikipedia.org/wiki/Object-relational_mapping [ref-act]: http://wikipedia.org/wiki/Active_record

		join_table (line 1100)
	
 
	
	

This uses alphabetical comparison to choose the name of the table.


Example: The joining table of users and roles would be roles_users,  because "r" comes before "u". Joining products and categories would  result in categories_products, because "c" comes before "p".
Example: zoo > zebra > robber > ocean > angel > aardvark

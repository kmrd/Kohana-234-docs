		xss_clean (line 277)
	
 
	
	

Clean cross site scripting exploits from string.


HTMLPurifier may be used if installed, otherwise defaults to built in method.  Note - This function should only be used to deal with data upon submission.  It's not something that should be used for general runtime processing  since it requires a fair amount of processing overhead.

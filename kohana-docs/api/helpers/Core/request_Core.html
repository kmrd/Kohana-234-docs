		static method preferred_accept (line 118)
	
 
	
	

Compare the q values for given array of content types and return the one with the highest value.


If items are found to have the same q value, the first one encountered in the given array wins.  If all items in the given array have a q value of 0, FALSE is returned.

	
    
				
  • return: string mime type with highest q value, FALSE if none of the given types are accepted
